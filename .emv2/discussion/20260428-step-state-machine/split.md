# 需求拆分 - 步骤状态自动推进

## 问题
子步骤完成后 `当前步骤` 没有自动推进到下一阶段。

## 涉及模块

| 编号 | 模块 | 说明 |
|------|------|------|
| 1 | `commands/result.md` | 提交结果时增加"状态推进"步骤 |
| 2 | `commands/stat.md` | 状态推进后 stat 能正确显示新步骤 |
| 3 | `project-spec.md` Meta | "当前步骤"字段需要支持子步骤粒度 |
| 4 | `workflows/hvr-workflow.md` | 验证通过的流程图中加入"推进当前步骤" |

## 推进层级

- **子步骤级**: S7-A 完成 → `当前步骤` 推进到 S7-B
- **主步骤级**: S7 全部子步骤完成 → `当前步骤` 推进到 S7-待验证 或 S8
