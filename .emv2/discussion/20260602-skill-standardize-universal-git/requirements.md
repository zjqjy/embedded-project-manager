## 关键设计原则（汇总）

1. **标准化选 skill-install 模板**（YAML frontmatter 规范）
2. **双轨制架构**（通用新结构 + 嵌入式原结构共存）
3. **嵌入式按需加载**（.em/embedded/ 可选模块 + 主入口 SKILL.md + 扩展 SKILL.md）
4. **目录改名 `.emv2/` → `.em/`**（保持命令前缀 `/em`）
5. **Git 自动化但用户把关**（AI 提议 + 用户确认 commit）
6. **嵌入式零迁移成本**（现有项目直接升级，文件位置不动）
