## 关键决策摘要

1. **标准化**: 顶部加 YAML frontmatter，不重构目录
2. **加载模式**: 单 SKILL.md 整加载（不拆分扩展）
3. **目录双轨**: 优先 `.em/`，回退 `.emv2/`
4. **嵌入式收纳**: 串口/烧录/编译/芯片学习 → `.em/embedded/`
5. **init 智能识别**: 自动检测 Keil/CubeMX/ESP-IDF/PlatformIO 特征
6. **Git 自动化但用户把关**: AI 提议 + 用户确认 commit；**禁止 push**
7. **零外部依赖**: CHANGELOG 用 EM 自写 Python 脚本
8. **完全向后兼容**: 现有 `.emv2/` 项目零配置升级
