# Split — S10: EM-SKILL 学习模式升级 v4.1

## 子功能清单

### F1: 主题 README 卡片规范
- 输入: 学习主题描述
- 输出: README.md (精简版 150-200 行)
- 验收: 包含钩子、3句话总结、概念图、架构图、代码、踩坑、Related Topics
- 依赖: 无

### F2: LPR 闭环工作流
- 输入: 功能描述
- 输出: L1-L5 阶段产物
- 验收: 每个阶段有明确产出文件和验收标准
- 依赖: F1 (L5 依赖 README 卡片)

### F3: 知识图谱索引
- 输入: 主题元数据
- 输出: _index.json + 知识图谱
- 验收: 支持主题关联查询、复用指数统计
- 依赖: 无 (可与 F4 并行)

### F4: 调研资料轻量管理
- 输入: 视频/文章/手册 URL
- 输出: bib.json + research/README.md
- 验收: 结构化引用、可追溯、不存大文件
- 依赖: 无 (可与 F3 并行)

### F5: 多格式构建脚本
- 输入: Markdown 源文件
- 输出: HTML / 口播稿 / 海报 / .skill
- 验收: 一键生成、暗色主题、Mermaid 渲染
- 依赖: F1 (需要 README 结构稳定)

### F6: 分发渠道矩阵
- 输入: 构建产物
- 输出: 多平台内容
- 验收: GitHub Pages 自动部署、视频脚本可用
- 依赖: F5

## 并行计划
- 第一波: F1, F3, F4 (独立)
- 第二波: F2 (依赖 F1), F5 (依赖 F1)
- 第三波: F6 (依赖 F5)
