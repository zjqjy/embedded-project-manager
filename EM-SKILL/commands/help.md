# 命令: /em help (帮助)

## 功能
显示帮助信息

## 触发
```
/em help [命令]
```

## 无参数时
显示所有命令列表和简介

## 有参数时
显示指定命令的详细帮助

## 示例

```
/em help           # 显示所有命令
/em help verify    # 显示 verify 命令帮助
/em help result    # 显示 result 命令帮助
/em help disc      # 显示 disc 命令帮助
```
