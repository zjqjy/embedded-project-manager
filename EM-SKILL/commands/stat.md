# 命令: /em stat (当前步骤)

## 功能
查看当前项目状态和下一步行动

## 触发
```
/em stat
```

## 执行流程

1. 读取 `.emv2/memory-log.md`
2. 读取 `.emv2/project-spec.md`
3. 解析「开发步骤状态」表，提取当前最大 S 编号
4. 生成状态摘要（含当前步骤编号）

## 输出格式

```
📍 当前步骤

项目: <项目名称>
步骤: S<编号>-<描述>
状态: <状态>

━━━━━━━━━━━━━━━━━━━━━━━━━━
下一步行动:
1. <下一步1>
2. <下一步2>

快捷命令:
- /em verify s<N>   # 验证
- /em disc          # 讨论
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 工作流推荐

根据当前步骤自动推荐命令：

| 当前步骤 | 推荐命令 |
|----------|----------|
| 开发中 | `/em verify <步骤>` |
| 待讨论 | `/em disc` |
| 等待硬件验证 | `/em verify` |
| 需要归档 | `/em arch` |
